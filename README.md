1. Akses Website
Buka Browser: Jalankan browser favorit Anda.
Kunjungi Website: Masukkan URL aplikasi.
2. Login ke Sistem
Masukkan Kredensial: Gunakan Nama Lengkap dan password (NISN) yang telah disediakan oleh admin untuk mengakses sistem.
Autentikasi: Setelah memasukkan data, klik tombol login untuk masuk ke dashboard.
3. Fitur untuk Voter
Pilih kandidat yang diinginkan dan berikan suara dengan satu klik.
4. Fitur untuk Admin
Kelola Data Voter:
Daftarkan siswa untuk digunakan sebagai voter, serta hapus data voter jika diperlukan.
Kelola Kandidat:
Tambah atau hapus informasi kandidat untuk memastikan data yang ditampilkan selalu terbaru.
5. Logout
Amankan Akun:
Setelah selesai menggunakan aplikasi, pastikan untuk logout agar akun tetap aman.
